Запуск ДраконТех без исполняемых файлов

Если вы не доверяете неизвестным исполняемым файлам или ваша архитектура не поддерживается, вы можете запустить ДраконТех из исходного кода без сборки.

  1. Скачайте исходный код ДраконТех.
  2. Установите NodeJS.
  3. Глобально установите npm-пакет electron: npm install -g electron
  4. Перейдите в папку с исходным кодом ДраконТех.
  5. Запустите файл runme или выполните следующую команду: electron src/main.js
