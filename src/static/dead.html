Drakon.Tech Service Shutdown

Drakon.Tech home

We regret to inform you that the Drakon.Tech online service will be shutting down permanently on December 7, 2024. After this date, all your projects will be deleted permanently, and you will no longer be able to access them through the platform.

To avoid losing any of your work, we strongly recommend that you save all source code you have generated using Drakon.Tech before December 7, 2024.

If you would like to continue using Drakon.Tech beyond the shutdown, it is possible to run the service on your own Linux server. You can find the necessary source code and instructions on GitHub here: https://github.com/stepan-mitkin/drakon.tech.

We apologize for any inconvenience this may cause and appreciate your understanding.

Best regards,
The Drakon.Tech Team
